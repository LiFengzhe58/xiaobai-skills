## What Sponsorship Supports

Sponsorship helps fund:

- maintaining the recommended beginner skill stack
- testing installs on more Codex environments
- improving the PowerShell installer
- writing beginner-friendly docs
- adding safer backup and restore workflows

## Notes

- Please only sponsor if this project actually saves you time.
- Sponsorship does not buy private user data, hidden priority, or unsafe
  behavior.
- If your organization wants to sponsor publicly, open an issue and we can add a
  public sponsor acknowledgement section to the README.
